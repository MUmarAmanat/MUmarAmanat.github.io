Single Data Source for all teams

Business is fed up with managing multiple data sources and syncing them all so that different team can fulfill their purpose. Business is looking for a single data platform that provides data according to team needs. Final solution was built by leveraging the power of Databricks and created a single source of truth known as Lakehouse architecture. Data Science, Machine learning teams, and data analyst can do their job by referring to the single data source. and do their job efficiently.
Worked on following technologies in this project:

  • Some of pipelines were built using Python.
  • From ingensting data to creating a final Lakehouse, PySpark proved to be a crucial tool.
  • Delta live tables pipelines were built using Databricks SQL.
  • Databricks used as a data processing tool.
  • All databases, and delta lakes connections keys were secured by Azure KeyVault
  • Solution was hosted on Microsoft Azure.
