Cloud Warehousing and scaled analytics with Azure Synapse

This client is from the production industry and is interested in doing analytics on its production data. Production data is actually machine data, this data should be incrementally ingested in data. After carefully analyzing the requirements, the final proposed solution was built using Azure ADF, and Synapse Analytics. Azure ADF serves as the primary data processing tool which gets data from the business's API.
Later cleaned data is stored in Azure Synapse. Datawarehouse is created on Azure Synapse, and reporting tables were also stored on Azure Synapse.
